# E-commerce Data Cleaning Project I

## Overview 
Cleaned a 1,200-row e-commerce orders dataset by handling missing values, duplicate records, and formatting inconsistencies.

## Tools Used
Microsoft Excel (Analysis Toolpak)

## What was Cleaned
- Filled 309 missing CouponCode values with "No Coupon"
- Verified zero duplicate OrderIDs across 1,200 records
- Standardized Date column to YYYY-MM-DD format
- Rounded UnitPrice and TotalPrice to 2 decimal places

## Files
- [Ecommerce dataset.csv](Ecommerce%20dataset.csv) — cleaned dataset
- [DECODESLAB_WEEK 1 CHANGE LOG.pdf](DECODESLAB_WEEK%201%20CHANGE%20LOG.pdf) — change log

## Dataset
14 columns: OrderID, Date, CustomerID, Product, Quantity, UnitPrice, ShippingAddress, PaymentMethod, OrderStatus, TrackingNumber, ItemsInCart, CouponCode, ReferralSource, TotalPrice
